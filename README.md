# sistem-informasi-catering-harian
CateringKu adalah platform web digitalisasi bisnis katering yang menghubungkan pelanggan dan pengelola. Pelanggan dapat memesan menu secara praktis melalui katalog digital, sementara Admin dapat mengelola data menu, memantau statistik pendapatan, serta melakukan verifikasi pembayaran dan pesanan secara terorganisir dalam satu sistem.
